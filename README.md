# FUTURE_PE_02 - AI Content Marketing Using UGC Ads

## Business Chosen
Glow Beauty Salon

## Objective
To create engaging User-Generated Content (UGC) style advertisements for a local beauty salon using prompt engineering techniques.

## Repository Structure

- README.md
- prompt.md
- output.md

## Project Description

This project demonstrates how AI can be used to generate authentic and engaging marketing content for social media platforms.

Using prompt engineering techniques, multiple prompts were designed to create UGC-style advertisements for Glow Beauty Salon across different platforms.

The generated content includes:
- Instagram Reel Ad Scripts
- Facebook UGC Ad Copy
- YouTube Shorts Ad Scripts
- Marketing Hooks
- Call-to-Action (CTA) Variations

## Prompt Engineering Strategy

- Assigned the AI a specific role (UGC Creator / Social Media Marketer).
- Defined the target audience.
- Specified the advertising platform.
- Used marketing frameworks such as Hook → Problem → Solution → CTA.
- Focused on creating authentic and relatable content.
- Generated platform-specific promotional material.
